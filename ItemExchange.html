In-Game Balance

Defining an "Item"

Keewano tracks game balance by defining the concept of an "item". An item is any countable element that can be given or taken within the game. Examples include:

  • Virtual currencies (e.g., gold coins)
  • Unique elements (e.g., a sword)
  • Game actions (e.g., number of moves)
Attention
This page covers virtual-to-virtual item exchanges (e.g., spending gold coins to buy a sword). For tracking items granted from in-app purchases (real money transactions), see In-App Purchases. For tracking items granted from watching ads, see Ad Revenue.

Item Names Identify Items Across All APIs

Items are identified by their name across all item-related SDK methods. Use the same item name whether the player earns it from gameplay, purchases it with real money, or receives it from watching ads.

Items Reset

To set or reset a specific item for the user, use KeewanoSDK.ReportItemsReset.

KeewanoSDK.ReportItemsReset is designed to both initialize items with default values and override existing values. This function is typically used during initial setup or in response to a support command to ensure the user's items are accurate and up-to-date.

Items Exchange

To track game balance, rewards, or lost items, use the KeewanoSDK.ReportItemsExchange. For example, a player spends 5 gold coins and a healing potion to get 1 vial of eternal life. Passing null as the to parameter means that the user lost the items (e.g., a broken sword or payment to pass a bridge), while passing null to the from parameter means the user gets a reward, such as collecting an item on the road or level win rewards.

Examples

Player Spends 5 Gold Coins and a Healing Potion to Get 1 Vial of Eternal Life

Keewano.Item[] fromItems = { new Keewano.Item("Gold Coin", 5), new Keewano.Item("Healing Potion") };
Keewano.Item[] toItems = { new Keewano.Item("Vial of Eternal Life") };
KeewanoSDK.ReportItemsExchange("Magic Store", fromItems, toItems);

Match3 Level Scenario

When the player enters a Match3 level, they lose 1 attempt and 1 bomb booster, their moves are reset to 5, they perform 3 moves (each reducing the moves count by 1), and finally win 50 gold coins and a rainbow booster.

// Step 1: Upon entering the level, deduct 1 attempt and 1 bomb booster
Keewano.Item[] initialLoss = { new Keewano.Item("Attempt", 1),
new Keewano.Item("Bomb Booster", 1) };
KeewanoSDK.ReportItemsExchange("Match3 Level Entry", initialLoss, null);
// Step 2: Reset moves to 5 at the start of the level
Keewano.Item[] movesReset = { new Keewano.Item("Moves", 5) };
KeewanoSDK.ReportItemsReset("Match3 Level Start", movesReset);
// Step 3: Player makes 3 moves, each costing 1 move
Keewano.Item[] moveCost = { new Keewano.Item("Moves", 1) };
KeewanoSDK.ReportItemsExchange("Match3 Move", moveCost, null);
KeewanoSDK.ReportItemsExchange("Match3 Move", moveCost, null);
KeewanoSDK.ReportItemsExchange("Match3 Move", moveCost, null);
// Step 4: Upon winning, reward the player with 50 gold coins and 1 rainbow booster
Keewano.Item[] winReward = { new Keewano.Item("Gold Coin", 50),
new Keewano.Item("Rainbow Booster") };
KeewanoSDK.ReportItemsExchange("Match3 Win", null, winReward);

User Balance Fix Reset by Support Team

In this scenario, the support team issues a command to fix a user's balance, resetting it to a predefined default state.

Keewano.Item[] balanceFix = { new Keewano.Item("Gold Coin", 100),
new Keewano.Item("Healing Potion", 2),
new Keewano.Item("Attempt", 3) };
KeewanoSDK.ReportItemsReset("Support Balance Reset", balanceFix);

Coffee Shop: Pay with Voucher or Credits + Stamp Card

A coffee can be paid with EITHER a Drink Voucher OR 5 Cafe Credits.
Every purchase adds +1 to a simple stamp card. At 10 stamps, the next coffee is free.

// Case A - Pay with a voucher, also add one stamp
Keewano.Item[] fromVoucher = { new Keewano.Item("Drink_Voucher") };
Keewano.Item[] toStamp = { new Keewano.Item("Stamp_Count") };
KeewanoSDK.ReportItemsExchange("Cafe_BuyCoffee_Voucher", fromVoucher, toStamp);
// Case B - Pay with cafe credits instead, also add one stamp
Keewano.Item[] fromCredits = { new Keewano.Item("Cafe_Credits", 5) };
KeewanoSDK.ReportItemsExchange("Cafe_BuyCoffee_Credits", fromCredits, toStamp);
// Case C — Pay with a combination: 1 voucher + 2 credits → 1 specialty coffee
Keewano.Item[] fromCombo = { new Keewano.Item("Drink_Voucher"), new Keewano.Item("Cafe_Credits", 2) };
Keewano.Item[] toSpecial = { new Keewano.Item("Coffee_Specialty") };
KeewanoSDK.ReportItemsExchange("Cafe_BuyCoffee_Combo", fromCombo, toSpecial);
// When Stamp_Count reaches 10 → consume 10 stamps and grant the free coffee
Keewano.Item[] spendStamps = { new Keewano.Item("Stamp_Count", 10) };
Keewano.Item[] freeCoffee = { new Keewano.Item("Coffee_Free") };
KeewanoSDK.ReportItemsExchange("Cafe_StampReward", spendStamps, freeCoffee);

Road Trip Day: Reset, Mid-Trip Changes, End-of-Day Settlement

// Start of the day: set a clear baseline for the trip-only items
Keewano.Item[] dayStart = {
new Keewano.Item("Trip_FuelUnits", 100),
new Keewano.Item("Trip_Snacks", 0),
new Keewano.Item("Trip_Credits", 30)
};
KeewanoSDK.ReportItemsReset("Trip_StartOfDay", dayStart);
// Drive a segment → use fuel
Keewano.Item[] fuelUse = { new Keewano.Item("Trip_FuelUnits", 15) };
KeewanoSDK.ReportItemsExchange("Trip_DriveSegment", fuelUse, null);
// Buy snacks → spend credits, gain snacks
Keewano.Item[] paySnacks = { new Keewano.Item("Trip_Credits", 3) };
Keewano.Item[] getSnacks = { new Keewano.Item("Trip_Snacks", 2) };
KeewanoSDK.ReportItemsExchange("Trip_BuySnacks", paySnacks, getSnacks);
// Pay a toll → spend credits (one-sided)
Keewano.Item[] payToll = { new Keewano.Item("Trip_Credits", 4) };
KeewanoSDK.ReportItemsExchange("Trip_PayToll", payToll, null);
// Find a coupon → gain credits (one-sided)
Keewano.Item[] couponGain = { new Keewano.Item("Trip_Credits", 2) };
KeewanoSDK.ReportItemsExchange("Trip_FindCoupon", null, couponGain);
// End of day: convert leftover trip credits into “savings” (persistent)
Keewano.Item[] settleFrom = { new Keewano.Item("Trip_Credits", 7) };
Keewano.Item[] settleTo = { new Keewano.Item("PiggyBank_Points", 7) };
KeewanoSDK.ReportItemsExchange("Trip_EndOfDay_Settle", settleFrom, settleTo);
